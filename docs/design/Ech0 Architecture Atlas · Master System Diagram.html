Master System Diagram · System Architecture

Ech0 Architecture Atlas / Single Go Binary · Self-Hosted Timeline Platform

A single synchronous request spine runs down the center (handler → service → repository → DB); the Agent subsystem and the event-driven subsystem hang off the left and right wings as peer subsystems; four foundations — config · tools · in-house libraries · local disk — sit beneath the binary. Infrastructure is placed where it actually acts, not stacked into one uniform grid.
