## 编辑器模式

Ech0 编辑器提供多种模式，通过模式面板切换：

| 模式 | 说明 |
| --- | --- |
| **Echo 编辑** | 默认模式，Markdown 编辑器 |
| **模式面板** | 显示可用的扩展和收件箱入口 |
| **扩展编辑** | 编辑音乐/视频/GitHub/网站链接等扩展卡片 |
| **图片管理** | 图片上传、布局选择、压缩开关 |
| **标签管理** | 为 Echo 添加或移除标签 |
| **收件箱** | 查看和管理收件箱消息 |

## 草稿自动保存

编辑器内建自动草稿保存机制，防止意外丢失内容。

### 工作原理

- 监听编辑状态的变化（文字、文件、扩展、标签等）
- 变化后 **600ms** 防抖自动保存到浏览器 `localStorage`
- 草稿有效期 **24 小时**，过期自动清除
- 页面关闭或隐藏时立即刷新保存

### 保存内容

草稿可保存文字与隐私设置、图片布局、附件、扩展内容、标签等。

### 恢复机制

下次打开编辑器时，若存在有效草稿，会弹出确认对话框选择恢复或清除。

> **提示：** 编辑已有 Echo 时通常不会覆盖草稿逻辑（以当前版本行为为准）。

## 图片智能压缩

上传图片时可开启「智能压缩」：

- **Safari**：多输出为 `image/jpeg`
- **其他浏览器**：多输出为 `image/webp`
- 在图片管理模式中通过开关控制；仅对本地上传生效。

## 上传配置

- 支持拖拽上传
- 单批与并发、重试等限制以当前实例界面说明为准
- S3 模式下可支持预签名直传
