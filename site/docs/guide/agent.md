title: 智能摘要
description: LLM Agent 配置与公开接口
---

在 **系统设置 → Agent** 中可启用「近期内容摘要」类能力：由大模型根据你最近的 Echo 生成一段简短总结，便于访客快速了解动态。

---

## 配置项（常见）

| 项       | 说明                                                                                       |
| -------- | ------------------------------------------------------------------------------------------ |
| 启用     | 总开关                                                                                     |
| 接口协议 | 按协议族区分，仅三项：**OpenAI 兼容**、**Anthropic**、**Gemini**                            |
| 模型名   | 如 `gpt-4o`、`deepseek-chat`、`claude-3-5-sonnet` 等，以厂商文档为准                        |
| API Key  | 访问密钥（勿泄露）                                                                          |
| Base URL | 自建或兼容网关地址；接 DeepSeek、Qwen、Ollama 等服务时在此填其接口地址                      |
| 提示词   | 可自定义系统提示，影响摘要风格与长度                                                        |

::: tip 协议而非厂商
这里选的是**接口协议**而不是具体厂商。DeepSeek、Qwen、通义、Ollama、各类自建兼容网关都走 **OpenAI 兼容** 协议——选它，再用 `Base URL` + `模型名` 指定具体实现即可。Anthropic（Claude）与 Gemini 走各自的原生协议。
:::

具体字段名以界面为准。

---

## 行为概要

- 会从近期 Echo 抽取上下文再调用模型。
- 可能有**缓存**或**单次合并请求**，避免短时间重复打爆 API（以实现为准）。
- 摘要就绪后可能通过**收件箱**等方式通知管理员。

---

## HTTP 接口（节选）

常见包括（路径以实例为准）：

| 方法    | 路径                  | 说明                                               |
| ------- | --------------------- | -------------------------------------------------- |
| GET     | `/api/agent/recent`   | 获取对外展示的摘要（是否匿名可访问取决于版本配置） |
| GET     | `/api/agent/info`     | 查询 Agent 是否启用等元信息                        |
| GET/PUT | `/api/agent/settings` | 读取或更新配置（通常需登录）                       |

完整定义见 **`/swagger/index.html`**。

## 费用与合规

调用第三方 LLM 可能产生费用；请遵守所选服务商条款，勿在提示词中放入密钥或隐私数据。
