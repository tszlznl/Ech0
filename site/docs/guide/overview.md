# 什么是 Ech0？

Ech0 是一款面向个人创作者的开源自托管发布平台。它以低资源占用、快速部署和长期可维护为核心，让你稳定发布想法、文字与链接，并持续掌握自己的数据与工作流。

## 产品亮点

- **轻量高效架构**：低资源占用与小体积镜像，适合个人服务器到 ARM 设备。
- **极速部署体验**：开箱即用 Docker 部署，从安装到运行一条命令即可启动。
- **自包含部署包**：提供完整二进制与容器镜像，无需额外依赖。
- **跨平台支持**：支持 Linux、Windows 与 ARM 架构设备（如 Raspberry Pi）。

快速开始请见 [安装部署](/docs/start/installation)。

## Storage & Data

- **统一存储抽象**：本地存储与 S3 兼容对象存储的挂载与管理。
- **数据主权**：内容与元数据由用户掌控。
- **迁移与备份**：支持迁移导入、快照导出与多种备份方式。

## Writing & Content

- **Markdown 写作**：内置编辑器与实时预览。
- **草稿自动保存**：编辑内容定期保存到本地，短期内可恢复。
- **标签与检索**：标签分类与过滤。
- **富媒体卡片**：链接、音乐、视频等扩展展示。

## Auth & Security

- **OAuth2 / OIDC**：便于接入第三方登录。
- **Passkey**：无密码登录能力。
- **访问令牌**：用于 API 与自动化集成。
