---
title: 访问令牌
description: 调用 API 与集成评论等场景
---

在 **系统设置 → 访问令牌** 中创建令牌，用于脚本、自动化或第三方系统以 **HTTP** 方式调用 Ech0 接口。

---

## 用法

在请求头加入：

```http
Authorization: Bearer <你的令牌>
```

具体哪些接口需要令牌、哪些可匿名，见实例 **`/swagger/index.html`**。

---

## 有效期

创建时可选择有效期档位（如数小时、一月、长期等，以界面为准）。**长期令牌**泄露风险大，请谨慎。

---

## 权限（Scope / Audience）

部分场景需要特定权限，例如：

- 一般 API 读写  
- **集成评论**接口可能要求 `comment:write` 及 `integration` **audience**（见 README 与 [评论系统](/docs/guide/comment)）

创建令牌时按向导勾选；不要用超级权限令牌做测试脚本并提交到公开仓库。

---

## 管理

- 令牌完整字符串通常在创建时**只显示一次**，请立即复制保存。  
- 列表中可随时**删除**即吊销。  
- 若怀疑泄露，应马上删除并换新。

---

## 与 OpenAPI

所有路径、请求体、响应码以部署实例上的 **Swagger** 为准；不同版本可能有字段差异。
