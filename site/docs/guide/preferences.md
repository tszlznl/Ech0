title: 偏好设置与用户资料
description: 面板中的系统 Logo、站点信息、页脚、头像与界面语言
---

管理后台里和 **站点外观、账号资料** 相关的项，主要分布在两个入口：

| 入口（侧边栏） | 路由 | 主要内容 |
| -------------- | ---- | -------- |
| **偏好设置** | `/panel/setting` | **系统设置**（站点 Logo、标题、服务地址、页脚等）+ **访问令牌** |
| **用户中心** | `/panel/user` | **个人头像**、用户名、密码、邮箱、**界面语言**（以及管理员可见的「用户管理」） |

下面按界面顺序说明各项含义与注意点。

---

## 偏好设置 → 系统设置

在 **偏好设置** 页顶部的 **系统设置** 卡片中，可配置全站展示与行为。点击 **编辑** 修改字段，最后点 **应用** 保存。

### 服务器 / 站点 Logo（站点图标）

- 用于 **未登录访客** 在首页顶部看到的主图标（圆形展示）。  
- 在编辑模式下点击 **更改**，上传一张图片（支持常见图片格式）；上传走与附件一致的存储逻辑（本地或已配置的 [对象存储](/docs/guide/s3)）。  
- **若用户已登录**，首页顶部通常会优先显示 **当前用户的头像**，而不是站点 Logo；要让访客或互联场景看到「站标」，请保持站点 Logo 已设置。

### 站点标题

- 浏览器标签页标题、站点元信息等处会用到（具体以当前版本为准）。  
- 可与「服务名称」区分：一个偏 **品牌/SEO**，一个偏 **展示名称**。

### 服务名称

- 对外展示用的实例名称（如时间线、互联等场景中的展示名）。  
- 可与站点标题相同或不同，按你的品牌习惯填写。

### 服务地址

- 填写你对外访问的 **完整 URL**，需带 **`http://` 或 `https://`**，例如 `https://memo.example.com`。  
- **务必填写正确**：部分能力（例如互联、头像/资源在其它实例上的展示）会依赖「实例地址」解析；填错可能导致他站无法显示你的头像或信息。  
- 若更换域名，请同步更新此处并检查 [常见问题](/docs/start/faq) 与 [互联](/docs/guide/federation) 相关说明。

### 页脚内容、页脚链接

- **页脚内容**：站点底部展示的短文案（可为空）。  
- **页脚链接**：可选，需带 `http(s)://`，用于页脚可点击跳转（如备案、关于页）。

### Meting API

- 音乐卡片等功能使用的 **Meting 解析服务** 地址，需带 `http(s)://`。  
- 未填写时可能使用内置默认；生产环境建议自建或选用可信地址，避免依赖不可控第三方。

### 自定义 CSS / 自定义 Script

- **自定义 CSS**：注入到前端的样式，用于微调主题、间距等。  
- **自定义 Script**：注入脚本（非编辑状态下界面以掩码形式显示，避免误触泄露）。  
- **安全提示**：此处内容会影响所有访客页面，**仅应由可信管理员** 编辑；错误脚本可能导致页面异常或安全风险（XSS）。修改前建议备份。

### 默认语言

- 影响 **未登录访客** 或 **未在个人设置里改语言** 的用户看到的默认界面语言（简中 / English / Deutsch 等，以界面选项为准）。  
- 已登录用户可在 **用户中心** 单独选择 **界面语言**，与全站默认可不一致。

### 允许注册

- 关闭后，新用户将无法通过注册入口创建账号（仅影响开放注册，与管理员手动创建用户等策略以当前版本为准）。

---

## 偏好设置 → 访问令牌

同一页下方的 **访问令牌** 用于为脚本、自动化、MCP 等签发 API 凭证，与「外观」无关。详见 [访问令牌](/docs/guide/accesstoken) 与 [MCP 接入](/docs/guide/mcp)。

---

## 用户中心 → 个人资料与头像

在 **用户中心** 顶部 **用户设置** 卡片中，管理 **当前登录账号** 自身信息（点击 **编辑** / **应用**）。

### 头像

- 点击 **更改** 上传图片，作为你在 **个人主页、评论、登录后首页顶栏** 等处的头像。  
- 上传后需 **应用** 保存个人资料变更；若仅上传未保存，以实际界面行为为准。

### 用户名、密码、邮箱

- **用户名**：登录名与展示名，修改规则以当前版本为准。  
- **密码**：留空可能表示不修改（以界面提示为准）；修改后请使用新密码登录。  
- **邮箱**：用于联系或找回等场景（若功能启用）。

### 界面语言

- 仅影响 **你本人** 登录后看到的界面语言，**不**覆盖全局「默认语言」对其他访客的影响。

---

## 小结

| 你想改… | 去哪里 |
| ------- | ------ |
| 站点 Logo、全站标题、服务地址、页脚、Meting、自定义样式/脚本、默认语言、开放注册 | **偏好设置** → 系统设置 |
| 个人头像、自己的用户名/邮箱/密码、自己的界面语言 | **用户中心** → 用户设置 |
| API 令牌 | **偏好设置** → 访问令牌 |

若某项在界面上的文案与本文档不一致，**以你部署实例上的界面与 OpenAPI（`/swagger/index.html`）为准**。
